Colores

La armonía de colores y el equilibrio hace que un diseño sea más o menos efectivo. ¿Pero qué colores tenemos que seleccionar? Por lo general, se recomiendan respetar una de estas tres reglas:

  1. Colores de la misma gama cromática; es decir, diferentes tonos de un mismo color.
  2. Colores complementarios; aquellos situados próximos en la gama cromática
  3. Colores opuestos: son aquellos situados justo en el lado opuesto de la gama cromática y que ayudan al contraste.

Es recomendable no abusar de los colores. Dos o tres colores es suficiente y siempre que sea posible es bueno tener un color predominante

Glosario

  1. <!DOCTYPE html>: debe ser la primer declaración en el documento (previo a <html>). <!DOCTYPE> es una instrucción para el navegador que indica la versión de HTML que se está utilizando.

  2. <html>: La etiqueta <html> le indica al navegador que lo que está interpretando es un documento HTML. Es la etiqueta contenedora de todas las otras etiquetas HTML (salvo del <!DOCTYPE>). Esta etiqueta se cierra al final del documento con </html>.

  3. <head>: Esta etiqueta se abre despues de la etiqueta <html>. Esta etiqueta contiene propiedades de la página como el titulo, los documentos de estilo, los documentos de funcionalidad etc. Dentro del <head> se pueden encontrar los siguientes elementos:

    • <title> (elemento obligatorio.)
    • <style>
    • <base>
    • <link>
    • <meta>
    • <script>
    • <noscript>
    la etiqueta <head> se cierra con </head>.

  4. <body>: Esta etiqueta se abre justo despues del </head>. En su interior se encuentran todos los elementos de la pagina que con los que el usuario podrá interactuar, ya sea visualmente o mediante funcionalidad extra. Se cierra con </body> al final del contenido de la pagina.

  5. Títulos

    • <h1></h1> - Título de mayor jerarquía
    • <h2></h2>
    • <h3></h3>
    • <h4></h4>
    • <h5></h5>
    • <h6></h6> - Título de menor jerarquía

  6. <p></p>: Define un párrafo. El navegador automaticamente agrega cierto margen ántes y después de cada párrafo. Este margen puede ser modificado con CSS.

  7. href: Es una propiedad de HTML que se usa en ciertas etiquetas para definir una URL a la que se quiere enlazar.

  8. <a></a>: Se utiliza para definir un hipervínculo. su propiedad más importante es el "href" donde se indica el enlace al que se desea hacer referencia.

    Modo de uso: <a href="http://www.facebook.com"> Facebook </a>

  9. <img></img>: Se utiliza para definir una imagen en HTML. Sus propiedades principales son src y alt.

    • src: Especifica la dirección de la imagen.

    • alt: Especifica un texto alternativo en caso que la imagen no se cargue correctamente.

  10. Listas Se ultilizan para armar listas en HTML. Las etiquetas utilizadas para señalar sus elementos son:

    • <ul></ul> - Etiqueta para empezar una lista desordenada.
    • <ol></ol> - Etiqueta para empezar una lista ordenada.
    • <li></li> - Etiqueta para especificar un elemento de la lista.

  11. <div></div>: Se utiliza para separar parte del codigo en un conjunto el cual tendra las mismas cualidades o pertenecera a una misma parte de la pagina.

  12. id: Se ustiliza para identificar un elemento particular del codigo de HTML para usarlo como referencia en caso de necesitarlo o darle diseño con CSS. No puede haber más de un elemento con el mismo id.

  13. class: Se utiliza para darle diseño a traves de CSS a uno o más elementos del codigo de HTML.

Glosario

En esta sección encontrarás propiedades útiles para darle estilo a tus páginas!

La sintaxis de CSS consiste de un selector, que indica el elemento que se va a modificar, propiedades, que indican lo que se modificará de dicho elemento y valores que indican el valor que se le dará a esa propiedad.

h1 {color:blue; font-size:12px;}

En el ejemplo anterior:

  • selector: h1
  • propiedades: color y font-size
  • valores: blue y 12px
    Selector de Elementos

  • Poniendo como selector un elemento de HTML (sea un h1, un p o un div), el codigo CSS modificará todos los elementos de ese tipo. El ejemplo anterior modifica TODOS los h1 del HTML.

  • Selector de id: Se utiliza para seleccionar UNICAMENTE el elemento de HTML que lleva ese id. De esta manera, no se modificaran otros elementos del mismo tipo.

  • Selector de clase: Se utiliza para seleccionar todos los elementos que tengan esa clase asignada.

    Algunas propiedades útiles son:

  1. font-family: Define la tipografía del elemento seleccionado. La propiedad font-family puede tener varias tipografías almacenadas, en caso de que el navegador no pueda mostrar alguna. Existen dos tipos de nombres de familia tipográfica:

    • Nombre de Familia: El nombre particular de un tipo de letra determinado. "arial", "times", "courier", etc

    • Familia Genérica: El nombre genérico de un tipo de letra."serif", "sans-serif", "cursive", "fantasy", "monospace"

    Primero ponemos el tipo de letra que queremos mostrar, luego una familia genérica en caso tal que el navegados no pueda mostrar la tipografía deseada. Todos los valores se separan por comas.

    Ejemplo: font-family: arial, sans-serif;

  2. font-size: Define el tamaño de la letra. Por defecto especificaremos los tamaños de letra en Pixels(px) pero existen otros valores que se pueden implementar.

  3. color: Define el color de un elemento. Comunmente, el valor del color lo pondremos en hexadecimal o rgb de la siguiente manera:

    • Hexadecimal - color: #A538FF;

    • RGB - color: rgb(165, 56, 255);

  4. background-color:Define el color de fondo de un elemento. Los colores se especifican de la misma manera que en la propiedad color

  5. width: Define el ancho de un elemento. El valor lo podemos especificar en Pixels(px) o en un porcentaje del elemento que lo contiene.

  6. Box-Model

    • margin: Define el margen de un elemento. Los valores se pueden dar en Pixels o en un porcentaje del elemento que lo contiene.

    • padding: Define cuanto se extiende un elemento en todas sus direcciones.Los valores se pueden dar en Pixels o en un porcentaje del elemento que lo contiene.

    En ambas propiedades se puede definir el lado especifico que se quiere modificar left, right, top, bottom

  7. Bordes

    • border-style: Define el tipo de borde que se quiere mostrar, dotted, dashed, solid... Puede contener hasta cuatro valores, uno para cada lado.

    • border-color: Define el color de los cuatro bordes. El color se puede especificar en Hexadecimal o en RGB. Puede contener hasta cuatro valores, uno para cada lado.

    • border-width: Define el grosor de los bordes. Puede ser definido por valores especificos Pixels o por los tres valores predefinidos thin, medium, thick. Puede contener hasta cuatro valores, uno para cada lado.

    Se puede utilizar únicamente la propiedad border: y especificar los tres valores separados por espacios.

Para más información sobre las propiedades detalladas anteriormente, no dudes en consultar esta pagina!

Glosario

Programar no es mas que dar instrucciones a una computadora, en un lenguaje que ella pueda interpretar y responder.

En el caso de JavaScript, y este curso, vamos a estar programando instrucciones para el navegador, es por ello que todos los programas que realicemos estarán vinculados directamente a una página web. A continuación veremos algunas de las instrucciones que podemos darle a nuestro navegador.

Vincular un archivo de JS con un archivo HTML

Una vez que tengamos creado nuestro archivo de JavaScript (.js) es muy fácil vincularlo con nuestro documento HTML.
Lo que debemos hacer es abrir una nueva etiqueta script y decirle que tipo de documento vamos a vincular (con el atributo type) y a donde se encuentra dicho documento (con el atributo src). A continuación tenemos un ejemplo de cómo se vería una vinculación genérica:

<script type = "text/javascript" src = "programa.js" > </script>

Variables

Una variable es un espacio de memoria en la computadora reservado para almacenar un valor de nuestro programa.

Nombramiento de Variables

    Existen ciertas reglas al momento de nombrar variables que debemos tomar en cuenta:

  • Los nombres de las variables pueden contener: Letras, números, guión bajo y signo pesos.
  • Los nombres de las variables no pueden empezar con un número.
  • Los nombres de las variables distinguen entre mayúsculas y minúsculas.
  • Los nombres de las variables NO pueden ser palabras reservadas del lenguaje.

Declaración de Variables:

Para declarar una variable nueva, basta con usar el prefijo var seguido del nombre de la variable. Si se va a asignar un valor a dicha variable al momento de declararla, quedaría de la siguiente manera: var x = 5; En caso que el valor vaya a ser asignado mas adelante, basta con declarar var x;

¿Qué puedo guardar en una variable?

En una variable puedo guardar varios tipos de cosas. En el ejemplo anterior, vimos como en una variable x se asignaba el valor 5. Este valor, es lo que conocemos como un "número entero" o int. Veamos que otros tipos de datos podemos almacenar, y como se vería escrito en código.

  • int o integer: este tipo de dato, es el que se encarga de almacenar números enteros (es decir, sin decimales). una variable que almacena un número entero se vería de la siguiente manera:

    var x = 5;

  • float: los valores del tipo float son todos aquellos valores numéricos que no sean enteros. Cualquier valor que lleve una coma decimal será almacenado con este tipo de variable. Hay que notar, que en programación no trabajaremos con coma decimal sino con punto decimal. De esta manera, la declaración de una variable del tipo float se vería de la siguiente manera:

    var x = 3.5;

  • string: Cuando hablamos de strings, estamos hablando de cadenas de caracteres. Si definimos un caracter como una letra, por ejemplo 'a' podemos decir que un string es un conjunto de caracteres, por ejemplo "albatros". De esta manera, podemos almacenar, en una variable una palabra o frase con el tipo de variable string. es importante notar que, al momento de declarar un string, el texto que queremos almacenar debe ir entre comillas. Por ejemplo:

    var nombre = "bienvenido a javascript" ;

Es importante notar que, si bien tenemos múltiples tipos de datos que podemos almacenar en la memoria, no es necesario indicarle al lenguaje que tipo de variable vamos a declarar. ¡JavaScript se encargará de asignar el tipo de variable que corresponda al tipo de dato que ingresemos automaticamente!
